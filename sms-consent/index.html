SMS Notification Consent

IntraSafe sends SMS messages to keep security teams coordinated during incidents. This page documents our opt-in consent process for A2P 10DLC compliance. For details on how we handle your phone number and SMS data, see our Privacy Policy.

Where consent is captured

Volunteers see the following checkbox on the account-creation page when accepting a phone invite:

By checking this box and submitting this form, I consent to receive text messages from IntraSafe for security alerts, shift reminders, and team coordination messages. Reply STOP to opt out. Reply HELP for help. Message & data rates may apply. Message frequency may vary.

The checkbox is unchecked by default. Users can create an account without checking this box. Only users who affirmatively check the box will receive SMS messages. Consent status and timestamp are stored per-user in our database for compliance auditing.

Consent checkbox — account creation form (Join with Token)

How users opt in

  1. An organization administrator invites a user via a unique invite link.
  2. The user opens https://app.intrasafe.app and creates an account using the invite link.
  3. During account creation, a dedicated SMS consent checkbox is displayed (unchecked by default, separate from Terms and Privacy acceptance).
  4. If the user checks the box and submits the form, their consent and timestamp are recorded in the database.
  5. Only users who checked the box receive SMS messages.
  6. Users who did not check the box can enable SMS later from their Profile settings in the app.

Automated message responses

OPT-IN CONFIRMATION

When a user opts in, they receive: "IntraSafe: You're opted in to receive security alerts, shift reminders, and team coordination messages. Reply HELP for help or email support@intrasafe.app. Message frequency may vary. Msg&data rates may apply. Reply STOP to opt out."

HELP RESPONSE

When a user replies HELP, they receive: "IntraSafe: Security volunteer coordination platform. For support, email support@intrasafe.app or visit https://intrasafe.app. Reply HELP for help. Reply STOP to opt out."

OPT-OUT CONFIRMATION

When a user replies STOP, they receive: "IntraSafe: You are unsubscribed and will receive no further messages."

Program details

  • Brand name IntraSafe
  • Message types Security alerts Shift reminders Team coordination Incident response
  • Message frequency Varies — typically 0–5 messages per day depending on active incidents and scheduled shifts
  • Audience Security volunteers and staff who have been explicitly invited to an organization by an admin. No cold outreach.
  • Opt-in method Voluntary checkbox at account creation (unchecked by default, not required to create an account)
  • Opt-out Reply STOP to any message. Users can also disable SMS in their profile settings.
  • Help Reply HELP or email support@intrasafe.app
  • Carrier disclaimer Message & data rates may apply
